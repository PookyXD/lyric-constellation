01

the idea

turns lyrics into stars. each line scored by emotion, mapped as a constellation unique to that song.

02

how it works

type any song. lyrics get pulled from Genius automatically. every single line gets scored on an emotional scale from -1.0 to +1.0. negative is dark. positive is light. each line becomes a star — the more emotional, the bigger it burns. connected across the song's journey as a constellation unique to that song.

03

the build

python backend. genius api. transformer emotion model. matplotlib constellation generator. fastapi serving it live. pure html css js on the front. crt theme built from scratch.
